*In Non-Persistent access mode of CSMA :*
- <span style="color:#fe3333">Before transmitting</span> the data,
	- <span style="color:#fe3333">Each node</span> must <span style="color:#fe3333">sense</span> the <span style="color:#fe3333">channel</span>,
	- <span style="color:#fe3333">If</span> the <span style="color:#fe3333">channel</span> is <span style="color:#fe3333">idle</span>,
		- It <span style="color:#fe3333">immediately sends</span> the <span style="color:#fe3333">data</span>.
	- <span style="color:#fe3333">Else</span>, the <span style="color:#fe3333">station waits</span> for a <span style="color:#fe3333">random time</span> (<span style="color:#ff00ff">not continuously</span>),
		- And <span style="color:#fe3333">when</span> the <span style="color:#fe3333">channel</span> is <span style="color:#fe3333">found</span> to be <span style="color:#fe3333">idle</span>,
			- it <span style="color:#fe3333">transmits</span> the <span style="color:#fe3333">frames</span>.
