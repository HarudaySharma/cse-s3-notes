>Consider the following relation:
>- `Report` (`S#, Sname, C#, Ctitle, Lname, Room#, Marks, Grade`)
>	- S# → Sname
>	- C# → Ctitle
>	- C# → Lname (Assuming one course is taught by only one instructor)
>	- Lname → Room# (Assuming each instructor has his/her room)
>	- S#, C# → Marks
>	- Marks → Grade
>	- S#, C# → Grade
